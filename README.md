# 🎨 ArtHub – Online Art Marketplace

ArtHub is a modern online art marketplace built with the MERN stack, allowing artists to showcase and sell their original artworks while providing buyers with a secure and user-friendly platform to discover, purchase, and manage art collections. The application features role-based dashboards, secure authentication, Stripe payment integration, and powerful artwork management tools.

# ✨ Key Features

* 🔐 JWT Authentication with Email/Password and Google Login
* 👥 Role-Based Access (Admin, Artist, Buyer)
* 🎨 Browse, Search, Filter & Sort Artworks
* 🖼️ Featured Artworks Section on Homepage
* 👨‍🎨 Top Artists Showcase
* 📂 Art Categories with Filtering
* 💳 Secure Stripe Payment Integration
* 💎 Subscription Plans (Free, Pro & Premium)
* 📊 Admin Analytics Dashboard with Charts
* 📈 Sales History for Artists
* 🖼️ Image Upload using ImgBB
* 📱 Fully Responsive Design
* ⚡ Loading Skeletons & Custom Error Pages
